# Druid CatBarShift

Automatically switches your action bar to page 2 when your Druid enters **Cat Form + Stealth (Prowl)** — and switches back to page 1 when you leave Stealth or Cat Form.

No configuration needed. Just put your Stealth abilities on action bar 2 and you're good to go.

---

## Features

- Automatic action bar switch to **page 2** on Cat Form + Stealth
- Automatic switch back to **page 1** when leaving Stealth or Cat Form
- Supports English (`Prowl`) and German (`Schleichen`) WoW clients
- Extremely lightweight — no UI, no frames, no memory overhead

## Setup

1. Install the addon into `World of Warcraft/_anniversary_/Interface/AddOns/`
2. Put your Stealth abilities (Pounce, Shred, Ravage, etc.) on **action bar 2**
3. Done

## Slash Commands

None — the addon works fully automatically.
